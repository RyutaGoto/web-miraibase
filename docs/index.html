MIRAI BASEへようこそ

函館で頑張るみんなの基地を目指して

個人的な作業やグループでの作業の場にしたり、共通の技術を用いる仲間とのコミュニケーションを行ったり、実際にものづくりをして経験を積んでみたり。MIRAIBASEは、現在函館で多方面に活躍する方たちや、これから活躍していく方たちの活動拠点となるべく函館市美原に設置された共同作業場です。

wifi、電源などの各種インフラに加え、プリンタ/スキャナ、ディスプレイやプロジェクター、また事務用品、会議室など、複数人で利用可能な設備もご用意しております。
